> click once to arm audio, then run with Ctrl+Enter. press Ctrl+Space (or the Functions button) to open the function browser - search, or pick a category then a function. each shows a description and example, and inserts at the cursor.
> files: Ctrl+O open · Ctrl+S save (writes back to the same file) · Ctrl+Shift+S save as · Ctrl+Z undo. the magenta dot = unsaved changes.
> push it darker: swap RolandTR909 for RolandTR808, raise the grit with .crush(4), bend the bass with .lpq(20), or drop the tempo with .cpm(124).
